✅ Work Fact or Fiction

True or False? Click to reveal the answer!

Statement 1: Work requires both force and motion in the same direction.

Correct! If force and displacement are not in the same direction, no work is done. Work = F × d × cos(θ)

Statement 2: Holding a box still requires work.

Correct! There is no displacement, so no work is done. Even though you're using energy, work is zero because the box doesn't move.

Statement 3: A heavier object always requires more work to move than a lighter object.

Correct! Work depends on force and distance, not just mass. A light object could require more work if moved over a greater distance with a greater force.

Statement 4: Work is always positive.

Correct! Work can be negative. Negative work occurs when force opposes motion, like friction opposing movement.

Statement 5: The SI unit of work is the Newton.

Correct! The SI unit of work is the Joule (J). The Newton is the unit of force.

Statement 6: Pushing a car sideways while it moves forward does work on the car.

Correct! Only the component of force in the direction of motion counts. A sideways push perpendicular to motion does zero work on forward motion.
